# ü§ñ Telegram Bot Builder

A **visual, no-code/low-code tool** for building Telegram bots using drag-and-drop blocks. Built with Python (Flask) and Blockly.

![Telegram Bot Builder Screenshot](docs/screenshot.png)

---

## ‚ú® Features

- **Visual Block Programming** - Drag and drop blocks to build bot logic
- **300+ Pre-built Blocks** - Events, actions, logic, loops, API calls, and more
- **Custom Block Builder** - Create your own blocks with custom Python code
- **Block Store** - Install community-created block packages (OpenAI, Firebase, etc.)
- **Live Preview** - Run and test your bot instantly
- **Project Management** - Save, load, and export projects
- **Python Library Manager** - Install/uninstall pip packages from the UI
- **Export to Python** - Get clean, production-ready Python code
- **SQLite Database** - Built-in persistent storage for your bots
- **Desktop App** - Optional Electron wrapper for standalone use

---

## üöÄ Quick Start

### Prerequisites
- Python 3.8+
- pip

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/telegram-bot-builder.git
   cd telegram-bot-builder
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**
   ```bash
   python app.py
   ```

4. **Open your browser**
   ```
   http://localhost:5000
   ```

## üß© Block Categories

| Category | Description |
|----------|-------------|
| **Bot Setup** | Initialize your bot with token |
| **Events** | Commands, messages, callbacks, media |
| **Actions** | Reply, send media, edit, delete messages |
